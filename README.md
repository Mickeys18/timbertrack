# TimberTrack Kenya 🪵

A specialized Inventory and Order Management System designed for timber yards to track wood stock, sales, and customer bookings.

## 🚀 Key Features
* **Inventory Tracking:** Manage stock for various wood types (Cypress, Pine, Bluegum, Gravillia, and Mahogany).
* **Product Management:** Track items like Cladding, Steps, Spindles, Doors, and Beds.
* **Order System:** Process customer bookings and generate receipts.
* **Sales Reporting:** View reports to track business performance.
* **Database Integration:** Connects to the `timbertracka` MySQL database.

## 🛠️ Tech Stack
* **Language:** PHP 7.4.33
* **Database:** MySQL (XAMPP Stack)
* **Frontend:** HTML5, CSS3, JavaScript

## 📋 Setup & Installation
1. Move the `timbertracka` folder to your `htdocs` directory.
2. Import the `timbertracka.sql` file into your phpMyAdmin.
3. Check `db_connect.php` to ensure database credentials match your local setup.
4. Access the system at `localhost/timbertracka`.

## 🇰🇪 Local Context
Designed for the Kenyan timber industry to streamline operations from order to delivery.
